# Jora: A package of Product Management skills that is personalised for my daily usage as a Product person

## Repository Layout
- `pm-skills/skills`: Source skill folders (`SKILL.md`, `references/`, `scripts/`, `assets/`)
- `pm-skills/bundles`: Packaged `.skill` bundles for distribution/import
- `pm-skills/tooling`: Automation utilities (Jira + Confluence helpers)
- `pm-skills/output`: Local output folder for generated artifacts
- `skills`: External reference repository kept as-is (not part of the PM publication set)

## Primary PM Skill Set
- `pm-skills/skills/strategic-memo-helper`
- `pm-skills/skills/jira-master`

See `pm-skills/README.md` for usage and required environment variables.
